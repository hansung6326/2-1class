* 실제로 뭘 할 수 있나?

  * 기능 구현 - 여러 파일에 걸친 새 기능도 자연어 한 문장으로 시작
  * 테스트 작성 - 테스트 없는 코드에 유닛 테스트 자동 추가
  * Git 자동화 - 커밋 메시지 작성, 브랜치 생성, PR 오픈까지
  * 리팩터링 - 코드 품질 개선, 중복 제거, 스타일 통일
* 실제 워크플로우 예시
  * 과제 코드 작성 -> 에러 발생 -> 에러 메시지 복붙 -> Claude가 파일 수정 -> 확인 후 승인
