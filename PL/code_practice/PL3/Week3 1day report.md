### Fortran(최초)

* Formular tanslation language(수식을 번역하는데 사용하는 언어)

* First high-level programming language(최초의 고급 언어)

* 1954, IBM, John Backus

* ##### 특징 : 효율성(빠른 실행), 정적(static) 메모리 관리

---



### Cobol(business oriented)

* Common business oriented language

* 1959 ~ 1960, Grace Hopper(최초의 여성 프로그래머)

* ##### 특징: 낮은 가독성(해석이 어려움), Record 도입, 선언부(declaration) + 실행부(execution)

* 중요성: 비즈니스 관련 시스템 개발에 많이 이용됨, 과거에 잘 나감

---



### Algol(mother language)

* Algorithmic language

* 1950년대 말 ~ 60년대 초

* 범용(general-purpose) 프로그래밍 언어

* Mother language(모태 언어): PL/I, Simula 67, Pascal, C, ...  

* ##### 특징: 블록(block) 구조, 매개변수/ 파라미터/ 인자 전달 //모든 언어가 이 특징을 다 따라 사용함(모태 언어인 이유)

---



### Pascal(교육용)

* 1971, Niklaus Wirth

* ##### 특징: 프로그래밍 교육용, 블록 구조, 단순함

---



### C(매우 중요한 언어)

* 1972, AT&T Bell Lab. , 개발자: Dennis Ritchie, Ken Thompson

* UNIX 운영체제(operating system) 개발에 이용됨

* ##### 특징: 단순, 간결, 효율적(빠른 실행), 포인터 데이터 유형(C언어가 중요한 언어인 이유)

* ##### 단점: 낮은 가독성, 약한 타입 검사, 데이터 캡슐화 지원 않음

---



### C++(C+ 객체 지향 개념)

* AT&T Bell lab. , Bjarne Stroustrup(개발자)

* Simula 67 + C + class(데이터 캡슐화를 지원하는 도구)

* ##### 특징: 객체지향언어, 데이터 캡슐화 지원

---



### C#(자바에 대응하기 위한 하나의 방편으로 개발됨)

* 2000, Microsoft

* .NET environment

* console application, winform application

* 자바를 대응하기 위해 나옴(C++보다 쉽게)

* 객체지향언어

* 컴포넌트(component) 기반 프로그래밍

* ##### 특징: 강한 타입 검사, 배열 경계 검사, 초기화하지 않은 변수 탐지, 자동 쓰레기 수집

  ##### 국제화:unicode, apps for host, embedded systems

* Software Development Kit(SDK)
  * C# compiler: csc
  * editor

* Visual Studio
* Main --> 프로그램 구조: 정적(static) 메소드, class 내부에 존재
* 컴파일: csc Hello.cs

---



### Java(hybrid 방식 언어)

* 1991, Sun Microsystems, James Gosling(개발자)

* ##### 특징: 하드웨어/ 소프트웨어 플랫폼에 독립적, C++보다 단순함, 분산 환경 지원

  ##### 객체지향언어, application, applet

* Java platform
  * Java Virtual Machine(JVM) + Java API
  * JVM: 바이트코드(인터프리터의 일종)를 실행하는 소프트웨어
  * JAVA API: 클래스와 인터페이스를 포함하는 '라이브러리'

---



### Perl

* CGI 프로그램 개발에 이용

* 텍스트 파일 처리에 강점

* 인터프리터 언어

* ##### 특징: 변수를 선언하지 않음, 배열 크기를 미리 선언하지 않음, 디폴트 초기값 제공

---



### Lisp

* List processor

* 함수형 언어

* AI programming language(인공지능 프로그래밍 언어)

* 1958, John McCarthy(개발자)

* Common Lisp(오랫동안 유행함)

* ##### 특징: 자동 기억 장소 관리, 동적 바인딩

* 프로그램 실행 = 연속적인 함수 호출
